# Material Symbols Rounded

Official 24 px SVG assets from Google Material Symbols, used by the SLM screens in this directory.

Reference an asset from a sibling `*.layout.md` file with vector CNL:

```md
Icon id play_icon 24 by 24 position 100 100 svg assets/icons/material-symbols-rounded/play_arrow.svg color #FFFFFF
```

Included icons: `add_circle`, `arrow_back`, `download`, `folder_open`, `forward_10`,
`play_arrow`, `redo`, `replay_10`, `skip_next`, `skip_previous`, `star`, `undo`,
`upload_file`, and `video_file`.

Source commit: `819d78680a849ceef4c78f863d8753e3160b7c89`
